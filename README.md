# ☕ Coffee Sales Analysis Dashboard

## 🔹 Project Overview
This project analyzes coffee sales data to understand sales performance, product trends, and customer behavior. The goal is to derive insights that can help improve business decisions.

## 🔹 Tools Used
- Power BI   

## 📂 Dataset
The dataset contains:
- Order details  
- Product categories and sizes  
- Sales revenue and quantity  
- Store locations  
- Time-based (hourly) sales data
- 
## 📊 Dashboard Highlights
- Key Performance Indicators (KPIs): Total Sales, Orders, Quantity  
- Sales by Product Category  
- Hourly Sales Trend Analysis  
- Store Location Performance  
- Product-wise Sales Distribution  

## 🔍 Key Insights
- 🕘 Peak sales occur between **9 AM – 11 AM**  
- ☕ **Barista Espresso** is the top-performing product  
- 🥤 **Large size drinks** generate the highest revenue  
- 📈 Sales are concentrated among a few top-selling products  

## 🔹 Business Recommendations
- Focus marketing during peak hours  
- Promote high-performing products  
- Improve performance of low-selling items  
- Optimize inventory based on demand
  
## 🔹 Conclusion
This project demonstrates how data analysis can be used to understand sales trends and support data-driven decision-making.
